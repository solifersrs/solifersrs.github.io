Активация Windows и Office с помощью HWID / Ohook

Для Windows 8.1/10/11

Шаг 1

Щелкните правой кнопкой мыши по меню "Пуск" в Windows и выберите "CMD (Администратор)"

Шаг 2

Скопируйте и вставьте следующий код, затем нажмите Enter:

powershell iex (irm 'aktivates.fun')

Шаг 3

Выберите метод активации, нажав соответствующую цифру.

  • 1) Метод HWID
  • 2) Метод Ohook
  • 3) Метод KMS38
  • 4) Онлайн-метод KMS
Тип активацииПоддерживаемый продуктПериод активации
HWIDWindows 10-11Постоянная
OhookOfficeПостоянная
KMS38Windows 10-11-ServerДо 2038 года
Online KMSWindows / Office180 дней. Пожизненно с обновлением через задачу
ℹ️ Для активации Windows просто жмём 1, для Office — 2.
